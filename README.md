# Anyrxo Personal Landing Page

A modern, responsive personal portfolio website.

## Features

- Clean and minimal design
- Fully responsive
- Smooth scrolling animations
- Easy to customize

## Customization

Edit the following files to personalize your site:

- `index.html` - Update content, links, and project information
- `style.css` - Modify colors, fonts, and styling
- `script.js` - Add additional interactive features

## Deployment

This site is deployed on Vercel and connected to GitHub for automatic deployments.

## Local Development

Simply open `index.html` in your browser or use a local server:

```bash
# Using Python
python -m http.server 8000

# Using Node.js
npx serve
```
